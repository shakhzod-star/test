Цель этого задания - оценить, как кандидат умеет обрабатывать большой объем данных и визуализировать их результаты.

В вашем распоряжении два CSV файла: "ascending" и "descending". Вам предстоит представить данные из этих файлов в виде линейного графика. 
Вы свободны в выборе JavaScript-библиотеки для визуализации, так же как и в выборе цветовой палитры. 
В предоставленном вам примере на скриншоте имеются линии "vertical" и "horizontal", однако в вашем задании необходимо сосредоточиться исключительно на линиях "ascending" и "descending".

В файлах CSV зафиксированы данные о смещении точек с интервалом в 12 дней начиная с 2019 года и заканчивая концом 2020 года. 
Каждая строка соответствует отдельной точке. Задача состоит в том, чтобы вычислить среднее значение смещений для всех точек по каждой дате и отобразить это на графике.

Ваш график будет отличаться от приведённого примера. 
Изменение исходных CSV файлов не допускается, но их использование в задании является обязательным.



Вам предстоит выполнить следующие шаги для решения задачи:

Извлечение данных: 

Откройте два предоставленных CSV файла, "ascending.csv" и "descending.csv". 
Каждый файл содержит набор данных, представляющих смещения на различные даты.

Обработка данных:

Просмотрите данные в файлах и обратите внимание на структуру: каждая строка соответствует различным точкам смещения, а каждый столбец соответствует определённой дате.
Для каждой даты (колонки) в файлах необходимо рассчитать среднее значение смещений всех точек (строк). 
Среднее значение - это сумма всех значений смещений для данной даты, делённая на количество точек.
Повторите расчёт среднего значения для каждой даты в обоих файлах.

Визуализация данных:

Подготовьте два набора данных для графика: один набор для файла "ascending.csv" и другой для файла "descending.csv".
Каждый набор должен включать даты и соответствующие им средние значения смещений.
Используйте JavaScript-библиотеку для построения линейного графика 
На графике должны быть две линии: одна для "ascending" данных и вторая для "descending" данных. 
Каждая линия будет представлять собой соединение точек средних значений смещений в хронологическом порядке.

Демонстрация результата:

Отобразите график таким образом, чтобы по оси X были расположены даты, а по оси Y — рассчитанные средние значения смещений.
Убедитесь, что график чёткий, даты легко читаемы, а линии различимы и точно отражают рассчитанные данные.
