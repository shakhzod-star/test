Основное задание:

Цель: Понять уровень вашей готовности к освоению новых технологий и проверить ваши навыки в JavaScript.

Задача: Ваша задача - создать веб-страницу, на которой будет отображаться карта города Ташкент, реализованная с использованием ArcGIS Maps SDK for JavaScript. Для Basemap использовать стиль navigation.
Вам необходимо интегрировать на карту маркер с координатами y: 41.342260, x: 69.336241, который соответствует расположению Министерства цифровых технологий. 
Маркер должен быть обозначен специальной иконкой (иконка работы), предоставленной в приложенных материалах, и символизировать место работы.

Вы можете использовать чистый JavaScript или применить любой фреймворк по вашему выбору.

Дополнительное задание с повышенной сложностью:

Усовершенствуйте маркер, добавив к нему всплывающее окно (pop-up), которое будет отображаться при нажатии на маркер. Всплывающее окно должно содержать надпись: "Работа: Министерство цифровых технологий".

Ресурсы для освоения и реализации задания:

Ознакомьтесь с основами ArcGIS Maps SDK for JavaScript: https://developers.arcgis.com/javascript/latest/
Установка и настройка SDK: https://developers.arcgis.com/javascript/latest/install-and-set-up/
Инструкция для изменения стили Basemap: https://developers.arcgis.com/javascript/latest/api-reference/esri-Basemap.html
Руководства по добавлению точек, линий и полигонов: https://developers.arcgis.com/javascript/latest/tutorials/add-a-point-line-and-polygon/
Инструкция по отображению всплывающих окон: https://developers.arcgis.com/javascript/latest/tutorials/display-a-pop-up/


Это задание предназначено для оценки вашей способности изучать новые библиотеки и ваших знаний в области JavaScript.
